## How to Write a Post

1. Create file: `_posts/YYYY-MM-DD-title.md`
2. Add front matter:
```yaml
---
layout: post
title: "Your Title"
date: 2026-03-20
author: QQ
tags: [topic]
---
```
3. Write in Markdown below
4. Push to GitHub
